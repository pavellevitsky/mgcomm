Main menu

 ******************************************************
 ** Welcome to MGComm | Main menu                    **
 ** ------------------------------------------------ **
 **  0  Exit                                         **
 **  1  Set GSM preffered network type               **
 **  2  Set WCDMA preffered network type             **
 **  3  Set LTE preffered network type               **
 **  4  Camp on LTE cell (switch to LTE before)      **
 ** ------------------------------------------------ **
 **  V  Show modem FW version                        **
 **  I  Show IMSI for both slots                     **
 **  N  Show Neighbour cells (GSM/WCDMA/LTE)         **
 **  P  Show Paging signals                          **
 **  S  Trigger power scan (GSM+WCDMA+LTE)           **
 **  G  Trigger power scan GSM                       **
 **  W  Trigger power scan WCDMA                     **
 **  L  Trigger power scan LTE                       **
 ** ------------------------------------------------ **
 **  UL transmission configuration                   **
 ** ------------------------------------------------ **
 **  A  Show valid AT$MGPHYCFG parameters range      **
 **  B  Read current parameters values               **
 **  C  Send AT$MGPHYCFG with modified values        **
 ** ------------------------------------------------ **
 **  R  Reset the device                             **
